# Summary

Life Engine is a free, open-source personal data sovereignty platform that combines a self-hosted backend hub — which normalises and unifies your data from external services (email, calendar, identity, cloud storage) into a single API you own — with a plugin-driven client that can be built and used on all possible platforms (Desktop, Web, Mobile (iOS, Android)) that starts as an empty shell and gains features only through plugins you choose to install. The goal is to give individuals full ownership and control over their personal data, identity, and digital life without requiring technical skills, replacing corporate dependence with convenience you actually own.
